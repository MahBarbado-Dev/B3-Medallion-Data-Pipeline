## Estrutura do Projeto

A estrutura de pastas inicial foi criada com a seguinte organização:

- `banco_sql/`: Destinado a arquivos relacionados a banco de dados (scripts SQL, modelagem, etc.).
- `dados_brutos/`: Destinado ao armazenamento dos dados originais, sem processamento.
- `notebooks/`: Destinado a análises exploratórias, testes de hipóteses e prototipagem (Jupyter Notebooks).
- `scripts/`: Destinado aos códigos em produção e scripts consolidados (ex: ETL em Python).
